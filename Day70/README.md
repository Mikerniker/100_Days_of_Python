Git Commands (Windows)
- ```git status```
- ```git init```
- ```git add .```
- ```git rm --cached -r .```  Use this to remove files from my staging area / undo the last step. 

To Create a GitIgnore File:
- ```touch .gitignore```
- ```code .gitignore```  (Open GitIgnore for Windows)
